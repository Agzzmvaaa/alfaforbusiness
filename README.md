# Кейс Альфа — материалы для репозитория

Статичный прототип и сопутствующие файлы для загрузки в Git (в т.ч. GitHub Pages).

## HTML-файлы (два разных макета)

1. **`index.html`** и **`alfa-msb-credit-concept-demo.html`** — один и тот же **интерактивный концепт**: роли, очередь согласований, пороги по суммам, «Подписать», «Отправить в Альфа-Банк», сохранение в `localStorage`.
2. **`alfa-msb-credit-current-ui-demo.html`** — **статичный** пошаговый макет «текущего» сервиса по номерам рисунков из кейса (рис. 1–15): без той же логики согласований, это витрина экранов для сравнения с концептом.

Откройте **`index.html`** для демо концепта (удобно для GitHub Pages).

Локально: двойной клик по `index.html` или через локальный сервер из этой папки:

```bash
python3 -m http.server 8080
```

Затем откройте http://localhost:8080/

## Git

```bash
cd alfa-case-git-bundle
git init
git add .
git commit -m "Initial commit: Alfa MSB credit case materials"
```

Подключите удалённый репозиторий и выполните `git push`.

Для **GitHub Pages**: в настройках репозитория укажите ветку и корень (root); главная страница — `index.html`.
