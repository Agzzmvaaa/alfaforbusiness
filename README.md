# Кейс Альфа — материалы для репозитория

Статичный прототип и сопутствующие файлы для загрузки в Git (в т.ч. GitHub Pages).

## Интерактивный прототип

- Откройте **`index.html`** в браузере (дубликат рабочего концепта для удобного деплоя).
- Альтернативные имена: `alfa-msb-credit-concept-demo.html`, `alfa-msb-credit-current-ui-demo.html`.

Локально: двойной клик по `index.html` или через локальный сервер из этой папки:

```bash
python3 -m http.server 8080
```

Затем откройте http://localhost:8080/

## Git

```bash
cd alfa-case-git-bundle
git init
git add .
git commit -m "Initial commit: Alfa MSB credit case materials"
```

Подключите удалённый репозиторий и выполните `git push`.

Для **GitHub Pages**: в настройках репозитория укажите ветку и корень (root); главная страница — `index.html`.
